下面是一份“傻瓜式”教程，手把手教你用 Docsify 搭一个**左侧可点击跳转的日记站**。  
全程 0 编程，会打字就能搞定，10 分钟上线。

---

### 1. 准备工具（只需 2 个）
| 工具    | 干嘛     | 下载地址                                     |
| ------- | -------- | -------------------------------------------- |
| VS Code | 写文档   | [官网直接装](https://code.visualstudio.com/) |
| Node.js | 本地预览 | [官网下一步到底](https://nodejs.org/)        |

装完 Node 后，**打开终端**（Win 按 `Win+R` → 输入 `cmd`）执行：

```bash
npm i -g docsify-cli
```

出现一堆文字不报错就 OK。

---

### 2. 5 秒建项目
在桌面新建文件夹 `diary`，进去后**直接空白处右键 → 用 VS Code 打开**。  
在 VS Code 里再开一个终端（菜单 `Terminal → New Terminal`），输入：

```bash
docsify init .
```

一瞬间会出现 3 个文件，别管它们是啥。

---

### 3. 让左侧菜单长出来
1. 在根目录（`diary` 文件夹）新建文件 `_sidebar.md`  
2. 把下面内容**原样复制**进去保存：

```markdown
* 我的日记
  * [2025-12-01](docs/2025-12-01.md)
  * [2025-12-02](docs/2025-12-02.md)
  * [2025-12-03](docs/2025-12-03.md)
```

3. 打开已有的 `index.html`，找到这一段：

```javascript
window.$docsify = {
  name: '',
  repo: ''
}
```

改成：

```javascript
window.$docsify = {
  name: '我的日记',
  loadSidebar: true,   // 打开左侧菜单
  autoHeader: true,    // 自动显示标题
  subMaxLevel: 2       // 支持二级标题
}
```

保存，搞定。

---

### 4. 写日记
1. 在根目录新建文件夹 `docs`  
2. 在 `docs` 里新建文件 `2025-12-01.md`，内容随便写：

```markdown
# 2025 年 12 月 1 日

今天学了 Docsify，超级简单！
```

3. 想写新日记 → 复制 `2025-12-01.md` → 重命名为 `2025-12-04.md` → 把日期和内容改掉  
4. 打开 `_sidebar.md`，照样子加一行：

```markdown
  * [2025-12-04](docs/2025-12-04.md)
```

保存即可，菜单会自动更新。

---

### 5. 本地预览
在终端里执行：

```bash
docsify serve .
```

终端会给出地址，一般是：

```
http://localhost:3000
```

浏览器打开，左侧点日期，右边就能跳转，完事！

---

### 6. 发到网上（选做）
把整 `diary` 文件夹直接拖到 GitHub 仓库 → 打开 GitHub Pages → 选 `main` 分支 → Save，  
几秒后你就能拿到一个 `https://你的用户名.github.io/仓库名` 的公开日记站，手机也能看。

---

### 7. 常用口诀
| 想干的事     | 口诀                                                  |
| ------------ | ----------------------------------------------------- |
| 新增一篇日记 | 新建 `docs/日期.md` → 在 `_sidebar.md` 照抄加一行     |
| 改站点标题   | 改 `index.html` 里的 `name:`                          |
| 换颜色主题   | 把官方主题 CDN 链接粘进 `index.html` 的 `<head>` 即可 |

---

就这样，**本地能写、能看、能跳转，上线也只需拖一次文件夹**。  
全程只动了 2 个文件（`_sidebar.md` 和 `index.html`），真正的傻瓜式。祝写日记愉快！
