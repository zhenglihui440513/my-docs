# 基本信息

25年11月；TII；作者来自阿联酋和印度的电气学院；

主题是储能系统的能量调度，削峰填谷与套利；

基本思路是预测加控制；

# 关键词

云储能；局部交易；源荷不确定性；单时隙交易匹配；

# 介绍

1. ES(储能)可以削峰填谷和套利

2. 每户配备ES之后都可以参加市场，但给每户家庭都配备ES：

- 经济不现实；无法保证可以利益回收；
- 分布式ES的协调难度大；
- 能源使用也有不确定性；固定容量的ES无法适应消费模式的多样性；
- 固定容量的ES在非峰期造成容量浪费；
- 逆变器的标称的有功无功功率加大了inefficient；

3.需要cloud ES，云平台远程管理，利用产电和用电的历史数据

- [4]是该作者在25年在TIA发表的考虑建筑热不确定性的CES能量管理论文；

- [5-10]分析了CES的技术可行性；[9-10]引入了P2P；
- [11-14]集中式market；[15]考虑隐私；
- [16-19]分布式market;
- 不足：鲁棒的能量共享；公平的能量管理；最优定价；

> p2p 距离会影响交易意愿

- [20-21]建立了CES的商业模型；

4. 本文内容

single round 匹配；multi-round 匹配；主要解决源荷两端不确定性；局部交易；agent；

将能量存储到allocated CES spaces；
